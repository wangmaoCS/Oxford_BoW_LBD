This demo is used to show our result in Table 1 of our paper

If the paper is accepted, we will public our e-mail address 
for communication.

If you want run this script, you should work in Linux platform 
as our code rely the Yael library released by Inria.

The public data directory should contains feature data of VGG
Oxford dataset. You can download them from the FTP of Inria,
ftp://ftp.irisa.fr/local/texmex/corpus/iccv2013/

The image data is from http://www.robots.ox.ac.uk/~vgg/data/oxbuildings/
You can download the oxford building image data and place them under 
directory oxbuild_images
